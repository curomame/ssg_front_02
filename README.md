### 설치

<br>

### 1.Clone the repository
    git clone git@github.com:curomame/ssg_front_02.git

### 2.Install NPM packages
    npm install

### 3.Server API 입력 in .env.development
    REACT_APP_TEST_URL 서버 Url
    REACT_APP_DISPLAY_IMG_URL 서버 로컬 이미지 Url 

<br>

### 어플리케이션 실행 
    npm run start
